Требуется разработать библиотеку для LRU Cache.
В данной библиотеке должен быть реализован кэш LRU, работающий в 2х режимах:
- С ограничением по максимальному количеству записей в кэше, но без времени. Все свыше определенного количества удаляется из кэша, в зависимости по времени использования (наиболее старые удаляются, каждое обращение к объекту обновляет время его использования)
- С ограничением по времени жизни каждой записи. Устанавливается максимальное время нахождения записи в кэше, без обращения к ней. Любое обращение к записи сбрасывает счетчик времени. В этом варианте ограничение на количество записей не применяется.
- С ограничением по количеству и по времени. Это комбинация первых двух вариантов.
Кэш должен быть потокобезопасным.
Кэш должен предоставлять методы для сохранения состояния на жесткий диск и для его восстановления с жесткого диска. Это требуется для восстановления состояния кэша при перезапуске программы.
Кэш должен иметь простой, интуитивно понятный и хорошо документированный интерфейс.
